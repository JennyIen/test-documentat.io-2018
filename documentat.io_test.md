[–ú–µ–∂–¥—É–Ω–∞—Ä–æ–¥–Ω—ã–π —Å—Ç–∞–Ω–¥–∞—Ä—Ç Unicode](http://unicode.org/standard/standard.html) –∑–∞–∫—Ä–µ–ø–ª—è–µ—Ç —ç—Ç—É –¥–æ–≥–æ–≤–æ—Ä–µ–Ω–Ω–æ—Å—Ç—å, –¥–µ–ª–∞—è –µ–µ –µ–¥–∏–Ω–æ–π –¥–ª—è –≤—Å–µ–≥–æ –º–∏—Ä–∞. –≠—Ç–æ –∑–Ω–∞—á–∏—Ç, —á—Ç–æ –≤ –ª—é–±–æ–π —Ç–æ—á–∫–µ –ø–ª–∞–Ω–µ—Ç—ã —Ç–µ–∫—Å—Ç, –∑–∞–∫–æ–¥–∏—Ä–æ–≤–∞–Ω–Ω—ã–π —Å–∏–º–≤–æ–ª–∞–º–∏ Unicode, –º–æ–∂–Ω–æ –¥–µ–∫–æ–¥–∏—Ä–æ–≤–∞—Ç—å ‚Äî –∏ –ø—Ä–æ—á–∏—Ç–∞—Ç—å.  
[–ö–æ–Ω—Å–æ—Ä—Ü–∏—É–º Unicode](http://unicode.org/consortium/consort.html) –Ω–µ —Å–æ–∑–¥–∞–µ—Ç –Ω–æ–≤—ã–µ –¥–∞–Ω–Ω—ã–µ, –∞ —Ç–æ–ª—å–∫–æ —Å–æ–±–∏—Ä–∞–µ—Ç —Å–∏–º–≤–æ–ª—ã –≤ —Ç–∞–±–ª–∏—Ü—É –∏ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–∏–∑–∏—Ä—É–µ—Ç –∏—Ö, –ø—Ä–∏—Å–≤–∞–∏–≤–∞—è –Ω–æ–º–µ—Ä. –°–µ–π—á–∞—Å (–Ω–∞ —Å–µ–Ω—Ç—è–±—Ä—å 2018) –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è Unicode –≤–µ—Ä—Å–∏–∏ [11.0.0](http://unicode.org/versions/Unicode11.0.0/), –∫–æ—Ç–æ—Ä—ã–π –±—ã–ª –≤—ã–ø—É—â–µ–Ω –≤ –∏—é–Ω–µ 2018 –≥–æ–¥–∞.  
## –û—Å–Ω–æ–≤–Ω—ã–µ –ø–æ–Ω—è—Ç–∏—è Unicode
### <a name="aa"></a>–ö–æ–¥–æ–≤–∞—è –ø–æ–∑–∏—Ü–∏—è –≤ Unicode
**–ö–æ–¥–æ–≤–∞—è –ø–æ–∑–∏—Ü–∏—è** (code point) ‚Äî —ç—Ç–æ –Ω—É–º–µ—Ä–æ–≤–∞–Ω–Ω–∞—è —è—á–µ–π–∫–∞ —Ç–∞–±–ª–∏—Ü—ã Unicode, –≤ –∫–æ—Ç–æ—Ä—É—é –ø–æ–º–µ—â–µ–Ω —Å–∏–º–≤–æ–ª.  
–ö–æ–¥–æ–≤—ã–µ –ø–æ–∑–∏—Ü–∏–∏ –≤ Unicode –∑–∞–ø–∏—Å—ã–≤–∞—é—Ç—Å—è –∫–∞–∫ `U+` [—à–µ—Å—Ç–Ω–∞–¥—Ü–∞—Ç–µ—Ä–∏—á–Ω—ã–π](https://ru.wikipedia.org/wiki/%D0%A8%D0%B5%D1%81%D1%82%D0%BD%D0%B0%D0%B4%D1%86%D0%B0%D1%82%D0%B5%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0_%D1%81%D1%87%D0%B8%D1%81%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F) –Ω–æ–º–µ—Ä —è—á–µ–π–∫–∏ –∏ –∏–º–µ—é—Ç –≤–∏–¥: `U+–•–•–•–•` (–¥–ª—è —è—á–µ–µ–∫ —Å –Ω–æ–º–µ—Ä–∞–º–∏ –æ—Ç 0 –¥–æ FFFF), `U+XXXXX` (–¥–ª—è —è—á–µ–µ–∫ —Å –Ω–æ–º–µ—Ä–∞–º–∏ –æ—Ç 10000 –¥–æ FFFFF) –∏ `U+XXXXXX` (–¥–ª—è —è—á–µ–µ–∫ —Å –Ω–æ–º–µ—Ä–∞–º–∏ –æ—Ç 100000 –¥–æ 10FFFF).  
*–ù–∞–ø—Ä–∏–º–µ—Ä, —Å–∏–º–≤–æ–ª ¬´!¬ª –∏–º–µ–µ—Ç –∫–æ–¥–æ–≤—É—é –ø–æ–∑–∏—Ü–∏—é `U+0021`, –∞ —Å–∏–º–≤–æ–ª ¬´üòÄ¬ª ‚Äî `U+1F600`.*  
### –ö–æ–¥–æ–≤–∞—è –æ–±–ª–∞—Å—Ç—å –≤ Unicode
**–ö–æ–¥–æ–≤–∞—è –æ–±–ª–∞—Å—Ç—å** (–∏–ª–∏ –∫–æ–¥–æ–≤–æ–µ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ) ‚Äî —ç—Ç–æ –æ–±–ª–∞—Å—Ç—å —Ç–∞–±–ª–∏—Ü—ã Unicode, –∫–æ—Ç–æ—Ä–∞—è —Å–æ–¥–µ—Ä–∂–∏—Ç –≤—Å–µ –≤–æ–∑–º–æ–∂–Ω—ã–µ –∫–æ–¥–æ–≤—ã–µ –ø–æ–∑–∏—Ü–∏–∏. –í—Å–µ–≥–æ –≤ –∫–æ–¥–æ–≤–æ–π –æ–±–ª–∞—Å—Ç–∏ 1 114 112 –∫–æ–¥–æ–≤—ã—Ö –ø–æ–∑–∏—Ü–∏–π: –æ—Ç 0 –¥–æ 10FFFF.  
–ö–æ–¥–æ–≤–∞—è –æ–±–ª–∞—Å—Ç—å —É—Å–ª–æ–≤–Ω–æ —Ä–∞–∑–±–∏—Ç–∞ –Ω–∞ 17 ¬´–ø–ª–æ—Å–∫–æ—Å—Ç–µ–π¬ª (plane) —Å –Ω–æ–º–µ—Ä–∞–º–∏ —Å 0 –ø–æ 16. –ö–∞–∂–¥–∞—è –ø–ª–æ—Å–∫–æ—Å—Ç—å —Å–æ–¥–µ—Ä–∂–∏—Ç 65 536 –∫–æ–¥–æ–≤—ã—Ö –ø–æ–∑–∏—Ü–∏–π. –ù–∞ —Å–µ–≥–æ–¥–Ω—è—à–Ω–∏–π –¥–µ–Ω—å –∑–∞–¥–µ–π—Å—Ç–≤–æ–≤–∞–Ω–æ —à–µ—Å—Ç—å (—Å 0 –ø–æ 5) –ø–ª–æ—Å–∫–æ—Å—Ç–µ–π, –Ω–æ –±–æ–ª—å—à–∞—è —á–∞—Å—Ç—å –∏—Å–ø–æ–ª—å–∑—É—é—â–∏—Ö—Å—è –¥–ª—è —Å–æ–≤—Ä–µ–º–µ–Ω–Ω–æ–≥–æ –ø–∏—Å—å–º–∞ —Å–∏–º–≤–æ–ª–æ–≤ Unicode —Ä–∞–∑–º–µ—â–µ–Ω–∞ –≤ –Ω—É–ª–µ–≤–æ–π.

–û—Å—Ç–∞–ª—å–Ω—ã–µ –ø–ª–æ—Å–∫–æ—Å—Ç–∏ –Ω–∞–∑—ã–≤–∞—é—Ç—Å—è –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã–º–∏ (Supplementary), –æ–Ω–∏ —Å–æ–¥–µ—Ä–∂–∞—Ç —Å–∏–º–≤–æ–ª—ã –∏—Å—Ç–æ—Ä–∏—á–µ—Å–∫–∏—Ö –ø–∏—Å—å–º–µ–Ω–Ω–æ—Å—Ç–µ–π –∏ –∫–∏—Ç–∞–π—Å–∫–æ–≥–æ –ø–∏—Å—å–º–∞, –Ω–µ–∫–æ—Ç–æ—Ä—ã–µ —É—á–∞—Å—Ç–∫–∏ —ç—Ç–∏—Ö –ø–ª–æ—Å–∫–æ—Å—Ç–µ–π –∑–∞—Ä–µ–∑–µ—Ä–≤–∏—Ä–æ–≤–∞–Ω—ã –¥–ª—è —á–∞—Å—Ç–Ω–æ–≥–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è.
